Assignment 9: Arrays

Assignment 9 focuses on bringing HTML elements to life by creating the ability to loop thru an array. Elements can contain info via JavaScript and are animated/ moved like a slideshow similar to a powerpoint or gif, giving more function to our websites. This assingment utilzes flex boxes.

Assignment 10: Classes

Assignment 10 focuses on bringing HTML elements to life by creating the ability to loop thru an array. Classes can contain info via JavaScript and we created a toy class containing a description and price for each item, giving more function to our websites. This assingment utilzes flex boxes.

Assignment 11: JSON

Assignment 11 focuses on bringing JSON elements to our sites thru use of JavaScript. JSON files can contain info via classes and things we implement via JS and we created a movie class containing a description and Title, and image for each film. This assingment utilzes flex boxes.
